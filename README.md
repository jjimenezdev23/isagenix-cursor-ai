# isagenix-cursor-ai

Static landing page build for the Isagenix Kickstart 90 campaign.

## Current setup

- `index.html` is the entry point for previewing the page.
- `styles.css` contains the CSS initialization: reset, design tokens, base typography, layout primitives, and initial button styling.
- The page shell is capped at the supplied 1440px site width.

## Design foundation

- Primary font: Nunito Sans.
- Core palette: turquoise, ice blue, dusk gray, cloud white, and magenta accent.
- Section backgrounds: cream and sky blue.
- Type scale follows the supplied Display XL through CTA/Button setup.
